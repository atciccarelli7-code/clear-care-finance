# Care Readiness pilot risk register

| Risk | Current control | Release condition |
| --- | --- | --- |
| Wrong medication or regimen branch | Exact selection; rivaroxaban hard stop; no generic enoxaparin rule | Product binding and clinical simulation approved |
| Conflicting final instructions | Staff reconciliation hard stop | One authoritative final plan and escalation path approved |
| Unsafe local emergency wording | Public candidate refers to organization-approved plan | Qualified review and local signoff complete |
| False completion | Derived status; viewing and printing do not count | Workflow tests and staff simulation complete |
| Barrier hidden by “education complete” | Barrier owner/action/status; unresolved stop blocks | Case-management and pharmacy workflow approved |
| Sensitive data leakage | Fixed choices, session-only state, no free text, aggregate analytics | Privacy/security review of institutional data flow complete |
| Inaccessible critical instructions | Text labels plus color; semantic controls; print separation | WCAG-focused evaluation and user testing complete |
| Stale labeling or handout | Source IDs, review date, version, open decisions | Monitoring, replacement, and recall process active |
| Public preview mistaken for patient handout | Repeated not-approved boundary and blank fields | Limited-pilot release artifact distributed only through controlled channel |
| Workflow burden causes workarounds | Minimal fixed choices and measured active time | Pilot burden threshold and support plan approved |

All twelve private clinical decisions remain open at this website checkpoint. This is an implementation candidate, not a clinical release candidate.
