# Medicare and Medicaid Guide Chapter Drafts

Community Acquired Finance  
Phase 3 chapter-draft package  
Last updated: 2026-07-06

This folder breaks the flagship guide manuscript into separate chapter drafts. Each chapter uses the same template:

- Direct answer
- Plain-English explanation
- Common misunderstanding
- Hospital/caregiver example
- Questions to ask
- Related site tools
- Source note

These drafts are not the final designed guide. Phase 4 should fact-check every claim, verify current-year dollar amounts, add missing official sources, and tighten boundaries before PDF design or public release.
