# Medicare and Medicaid Guide Print Layout Package

Community Acquired Finance  
Phase 5 draft print/PDF layout  
Last updated: 2026-07-06

## Status

Draft layout package only. Do not distribute as the final guide yet.

The Phase 4 fact-check report still requires added official sources for dual eligibility, Medicare Summary Notices, EOB/billing details, observation notices, appeals, Medicare Advantage authorization, and state Medicaid/LTSS verification before public PDF release.

## Purpose

This folder defines the print-friendly structure for **The Hospital Family Guide to Medicare, Medicaid, Rehab, and Long-Term Care**.

It creates a plain visual hierarchy for the final guide without adding public site routes, creating a final PDF, publishing a landing page, or adding monetization.

## Files

- `print-layout-spec.md` - page architecture, visual hierarchy, QR placeholder rules, and print standards.
- `print-template.html` - draft print-ready HTML layout template with placeholder pages and print CSS.
- `print-release-checklist.md` - checklist to complete before generating or distributing a PDF.

## Visual hierarchy

The guide should be easy to skim under stress.

Use this hierarchy throughout:

1. Chapter title
2. Direct answer box
3. Plain-English explanation
4. Common misunderstanding callout
5. Hospital/caregiver example
6. Questions to ask checklist
7. Related site tools with QR placeholder
8. Source note

## QR placeholder rule

Use visible placeholders until the public landing page and final tool URLs are tested.

Placeholder text:

`QR PLACEHOLDER - DO NOT PRINT FINAL UNTIL URL IS LIVE`

## Non-goals for this phase

- No final PDF export.
- No public guide download.
- No landing page.
- No QR code generation.
- No design-heavy branding.
- No ads or affiliate blocks.
- No insurer rankings or sales framing.

## Next phase after this

Close source gaps, then create the final edited manuscript with endnotes. Only after that should the designed PDF and public landing page be generated.
