# Print Release Checklist

Guide: **The Hospital Family Guide to Medicare, Medicaid, Rehab, and Long-Term Care**  
Status: Draft checklist  
Last updated: 2026-07-06

Do not distribute the final PDF until every item below is complete.

## Source and fact-check requirements

- [ ] Add official dual-eligibility coordination source.
- [ ] Add official source for full vs limited Medicaid assistance.
- [ ] Add official Medicare Summary Notice source.
- [ ] Add official EOB/billing source or revise EOB language to be plan-document dependent.
- [ ] Add official observation notice / beneficiary rights source.
- [ ] Add official Medicare appeals source.
- [ ] Add official Medicare Advantage appeals / prior authorization source.
- [ ] Add state Medicaid sources for any state-specific appendix.
- [ ] Recheck all current-year dollar amounts.
- [ ] Remove or soften any statement that sounds like a personalized coverage determination.

## Editorial requirements

- [ ] Every chapter uses the same structure.
- [ ] Every chapter includes a source note.
- [ ] Every chapter includes practical questions to ask.
- [ ] No chapter gives medical, legal, insurance, tax, Medicaid planning, or individualized financial advice.
- [ ] No insurer rankings.
- [ ] No plan recommendations.
- [ ] No affiliate language.
- [ ] No sales copy.
- [ ] No hospital/employer/government endorsement language.

## Layout requirements

- [ ] US Letter size.
- [ ] Black-and-white printable.
- [ ] Body text no smaller than 10.5 pt.
- [ ] Source notes no smaller than 8.5 pt.
- [ ] Headings are easy to scan.
- [ ] Long paragraphs are broken up.
- [ ] Callout boxes do not split awkwardly when possible.
- [ ] Worksheets are mostly one page each.
- [ ] Footer includes educational-only language.
- [ ] Last updated date appears clearly.

## QR and link requirements

- [ ] Guide landing page exists.
- [ ] Guide landing page works on mobile.
- [ ] Download button works.
- [ ] `/medicare-care-costs#cost-estimator` is tested.
- [ ] `/tools/eob-to-bill-match-checker` is tested, or replaced with a live tool route.
- [ ] `/tools/out-of-pocket-max-estimator` is tested, or replaced with a live tool route.
- [ ] All QR codes are generated from tested live URLs.
- [ ] QR codes are scanned from a printed test page.
- [ ] No QR placeholder remains in the final PDF.

## PDF preflight requirements

- [ ] Export draft PDF.
- [ ] Render PDF pages to images.
- [ ] Check for clipped text, overlapping boxes, unreadable footnotes, broken glyphs, and bad page breaks.
- [ ] Print at least one black-and-white test page.
- [ ] Test on mobile PDF viewer.
- [ ] Confirm file name and metadata.
- [ ] Final PDF is saved under the chosen public guide path only after approval.

## Distribution requirements

- [ ] Final PDF includes disclaimer.
- [ ] Flyer includes disclaimer.
- [ ] Handout includes disclaimer.
- [ ] Social posts avoid fearmongering and sales claims.
- [ ] Materials do not imply endorsement by any hospital, insurer, Medicare, Medicaid, CMS, state agency, employer, or professional organization.
